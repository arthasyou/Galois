# 用最少的概念讲伽罗瓦的“思想”（不走后人的证明路线）

这组笔记只想做到一件事：**用最简单的方式让你抓到伽罗瓦式的核心感觉**：

> 关注的对象不是公式推导，而是“允许的对称/互换方式”如何形成结构、如何分层。

你可以把它理解成：在进入群论课程之前，先知道“这门课的大图景在干什么”。

## 这份笔记讲什么？

- 把“置换/对称”当作主角：我们研究的是“允许哪些互换”，而不是研究根的具体数值。
- 用“切割/分块”的直觉讲结构：为什么有的结构能继续分层，有的会卡在某一步。
- 尽量少概念：优先用例子、数量、对照（如 $S_3/S_4/S_5$）给出感觉。

## 这份笔记不讲什么？

- 不给严格证明，也不替代群论教材的定理链。
- **不证明“五次方程不能用根式解”**：这属于后人完善后的系统结论，需要一整套更正式的语言与证明。
- 我个人也不在这里强行建立“结构卡住 ⇔ 一定解不出”的对应关系；本笔记只负责把“结构如何分层”讲清楚。

## 阅读顺序

1. `00_根置换的表示方法.md`：先把“互换/置换”画出来（非常直观）。
2. `01-根的置换.md`：认识一些常见的互换类型（只用来分组，不要求背数量）。
3. `02-matrix.md`：介绍一个非常好用的“第一把刀”：把互换分成两类（偶/奇）。  
   - 如果你不想碰行列式，只看“交换次数的奇偶”那条路线即可。
4. `03-切割原则.md`：解释“刀”究竟是什么：不是随便分类，而是一条**与操作相容**的规则。
5. `04-判断是否可以再分解.md`：用 `S3/S4/S5` 做对比，展示“有时能切第二刀，有时只能切一刀”的直觉原因。

## 术语（尽量少）

- **置换**：把元素重新排列；只关心“谁被送到谁的位置”。
- **允许的互换 / 对称性**：在当前规则下，哪些置换被视为“允许发生”的。
- **一把刀（切割规则）**：把置换分成若干等大的块；关键是这把刀不能是“凑数分堆”，而要能支撑后续的分层讨论。
